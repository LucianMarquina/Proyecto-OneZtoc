# Inventario 360

**Inventario 360** es una solución móvil integral diseñada para optimizar y simplificar la gestión de inventarios. Con una interfaz moderna y fluida, esta aplicación permite a los usuarios mantener el control total de sus existencias en tiempo real.

---

## Características Principales

*   **Gestión en Tiempo Real**: Control total sobre entradas, salidas y stock actual.
*   **Sincronización con Odoo**: Integración nativa y robusta con el ERP Odoo para asegurar la consistencia de los datos.
*   **Interfaz Intuitiva**: Diseñada pensando en la experiencia de usuario para facilitar operaciones rápidas.
*   **Escalable y Segura**: Construida con tecnologías modernas para garantizar rendimiento y seguridad.

---

## Tecnologías Utilizadas

Este proyecto ha sido desarrollado utilizando un stack tecnológico robusto y moderno:

### Frontend (Móvil) and Backend

*   **[Odoo](https://www.odoo.com/)**
*   **[Python](https://www.python.org/)**

---

## Integración Odoo

La aplicación utiliza una conexión directa con la API de Odoo (XML-RPC / JSON-RPC) para:
1.  Autenticar usuarios del sistema ERP.
2.  Leer y escribir registros de productos e inventario.
3.  Sincronizar movimientos de stock al instante.
